# 🏠 MLOps House Price Prediction

This project is a full end-to-end Machine Learning application that predicts house prices using a trained model, a REST API, and a user-friendly interface.

---

## 🚀 Live Demo

👉 https://mlops-project-1-3hei.onrender.com

---

## 🧠 Features

- Machine Learning model (Random Forest)
- REST API built with FastAPI
- Interactive web UI for users
- Dockerized application
- Cloud deployment on Render
- MLflow integration for experiment tracking
