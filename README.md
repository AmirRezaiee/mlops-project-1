# mlops-project-1
# 🏠 MLOps House Price Prediction

This project is a full end-to-end Machine Learning application that predicts house prices using a trained model, a REST API, and a user-friendly interface.

---

## 🚀 Live Demo

👉 https://mlops-project-1-3hei.onrender.com

---

## 🧠 Features

- Machine Learning model (Random Forest)
- REST API built with FastAPI
- Interactive web UI for users
- Dockerized application
- Cloud deployment on Render
- MLflow integration for experiment tracking

---

## 🏗️ Project Structure

mlops-project-1/
│── src/ # API and training code
│── models/ # Saved ML model
│── data/ # Dataset (if used)
│── Dockerfile # Container configuration
│── requirements.txt
│── README.md


---

## ⚙️ How It Works

1. User enters input values (income, rooms, etc.)
2. FastAPI receives the request
3. Model processes the data
4. Prediction is returned to the user

---

## 🧪 Technologies Used

- Python
- FastAPI
- Scikit-learn
- Docker
- MLflow
- Render (Cloud)

---

## 💡 Example Use Case

Users can input housing features such as:

- Median income
- Number of rooms
- Population
- Location

and receive an estimated house price.
